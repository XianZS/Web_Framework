# 2、思路

（1）计算平均值avg

（2）计算“平均值-差值”数组

（3）计算“平均值-差值-前缀和”数组

（4）排序，找出“平均值-差值-前缀和”数组的中间mid

（5）累加“平均值-差值-前缀和”数组每个元素与mid的abs和

# 3、代码

```python
n=int(input())
nums=list()
for _ in range(n):
	nums.append(int(input()))
# print(nums)
avg=sum(nums)//n
# print(mid)
# 平均数差值数组
for x in range(n):
	nums[x]-=avg
# 平均数差值前缀和数组
for x in range(1,n):
	nums[x]+=nums[x-1]
nums.sort()
res=0
# 计算平均数差值前缀和数组的中间数值
mid=nums[n>>1]
for cho in range(n):
	res+=abs(nums[cho]-mid)
print(res)
```
